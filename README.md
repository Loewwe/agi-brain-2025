# AGI-Brain v1

**AGI-контейнер (мозг)** — автономная система, которая наблюдает, анализирует, учится и советует Владельцу.

## Высший закон

> Максимизировать долгосрочную пользу и безопасность для Владельца в рамках законов, этики и заданных ограничений.

## Уровни автономии (v1)

| Level | Режим | Описание |
|-------|-------|----------|
| L0 | Read-only | Анализ, отчёты, рекомендации |
| L1 | Propose | Генерация патчей/планов без применения |
| L2+ | Apply | **Заблокировано в v1** |

## Структура

```
src/
├── api/          # REST API endpoints
├── configs/      # YAML-профили и политики
├── world/        # Модель мира и world_snapshot
├── perception/   # Адаптеры: трейдинг, web, docs
├── memory/       # Эпизодическая и семантическая память
├── planning/     # LLM-оркестратор (Brain)
├── tools/        # Реестр инструментов
├── learning/     # Самообучение
└── safety/       # ACL, audit, laws
```

## Запуск

```bash
# Установка зависимостей
pip install -e ".[dev]"

# Запуск тестов
pytest tests/ -v

# Запуск API
uvicorn src.api.main:app --reload
```

## Конфигурация

См. `src/configs/` для настройки:
- `owner_profile.yaml` — профиль Владельца
- `autonomy_levels.yaml` — уровни автономии
- `learning_sources.yaml` — белый список источников
- `tools_registry.yaml` — реестр инструментов
